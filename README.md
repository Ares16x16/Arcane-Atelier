# Arcane Atelier

Unity 6 workshop slice for `Arcane Atelier`.

## Open

Open the repository root `D:\coding_workspace\arcane-atelier` as a Unity project.

On first successful editor import, the workshop bootstrap auto-generates:

- `Assets/ArcaneAtelier/Workshop/Generated/*`
- `Assets/Scenes/SpellAssemblyScene.unity`

## Scene

Primary playable scene:

- `Assets/Scenes/SpellAssemblyScene.unity`

## Current scope

Implemented in this repository:

- Spell Assembly / workshop grid
- node placement, removal, replacement, and rotation
- directional resource flow
- three baseline production chains
- workshop inventory and prepared card output
- reward integration hooks
- battle handoff payload contract
